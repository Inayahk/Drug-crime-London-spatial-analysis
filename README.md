# Drug-crime-London-spatial-analysis

This repository contains the code, datasets and outputs used for the Principles of Spatial Data Science coursework 2 assignment exploring the spatial and temporal patterns of drug-related crime across London.

## Repository structure

### raw_data
The Original downloaded crime datasets, which includes additional offence catergories and records with missing data.

### altered_crime_data
The crime data that has been altered and cleaned by removing missing data and irrelavent catergories.

### Disparity_data
This is the csv file of disparity data.

### Crime_code
The notebook used for spatial analysis.

## Data sources

### Metropolitan Police crime data
Crime data was downloaded from:
https://data.police.uk/data/archive/

Files downloaded:
January 2018
January 2021
January 2024

The drug-related offences for 2015, 2018 and 2021 were extracted from the February Metropolitan Police outcomes CSV files contained within these archives.

### English Indices of Deprivation 2019
The deprivation data was downloaded from:
https://www.gov.uk/government/statistics/english-indices-of-deprivation-2019

Dataset used:
File 7: All ranks, deciles and scores for the indices of deprivation, and population denominators
